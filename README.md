Set Up a Discord Bot (One-Time)

Go to Discord Developer Portal.
Create a new Application > Bot tab > Add Bot.
Under Privileged Gateway Intents, enable Server Members Intent (if needed).
Copy the Bot Token (keep it secret!).
In your Discord server: OAuth2 > URL Generator > Scopes: bot > Permissions: Create Instant Invite > Paste URL to invite bot to server.
Note your Server ID (Guild ID) and Channel ID (e.g., general channel: Right-click channel > Copy ID, with Developer Mode enabled in Discord settings).
