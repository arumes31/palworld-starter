1. Datenverarbeitung auf dieser Website

Diese Website verarbeitet Daten zur Bereitstellung der Palworld Serversteuerung. Die Datenverarbeitung erfolgt auf Grundlage von Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse an der technischen Bereitstellung, Systemsicherheit und Missbrauchsabwehr).

2. Bereitstellung von Spieldaten (Palworld API)

Zur Statusanzeige werden aktuelle Spielerdaten (Spielernamen und Charakter-Level) live von den verwalteten Palworld-Servercontainern abgefragt und auf dieser Webseite öffentlich angezeigt. Diese Daten werden nicht dauerhaft gespeichert, sondern bei jedem Seitenaufruf direkt über die Schnittstelle des laufenden Spielservers geladen.

3. Cookies und Sitzungsdaten

Wir verwenden ausschließlich technisch notwendige, verschlüsselte Session-Cookies. Diese enthalten keine personenbezogenen Tracking-Daten, sondern dienen der Speicherung Ihres Captcha-Status, Ihrer Sprachpräferenz sowie dem Schutz vor CSRF-Angriffen (Cross-Site Request Forgery). Diese Cookies werden gelöscht, wenn Sie Ihren Browser schließen oder die Sitzung abläuft.

4. IP-Adressen und Server-Logs

Bei jedem Zugriff auf diese Webseite erfasst unser Webserver automatisch technische Daten. Dazu gehört auch Ihre IP-Adresse. Diese Erfassung ist technisch notwendig, um die Webseite auszuliefern, Angriffe abzuwehren und Missbrauch (z. B. durch Bots) zu verhindern. Logs werden nach gesetzlichen Fristen automatisch gelöscht.

5. Drittanbieter-Ressourcen (Google Fonts)

Diese Seite bindet Schriftarten („Google Fonts“) des Anbieters Google LLC ein. Um diese Schriftarten anzuzeigen, baut Ihr Browser eine Verbindung zu Google-Servern auf, wodurch Ihre IP-Adresse an Google übertragen wird. Weitere Informationen finden Sie in der Datenschutzerklärung von Google unter: policies.google.com/privacy.

1. Data Processing on this Website

This website processes data to provide the Palworld Server Control interface. Data processing is based on Art. 6 (1) (f) GDPR (legitimate interest in technical delivery, system security, and abuse prevention).

2. Display of Game Data (Palworld API)

To display active server status, current player information (player names and character levels) is queried in real-time from the running Palworld container and shown publicly. This data is not stored persistently; it is fetched dynamically upon each page load directly from the game's REST API.

3. Cookies and Session Management

We use only strictly necessary, encrypted session cookies. These do not store personal tracking profiles; they are used solely to remember your captcha resolution, language toggles, and to protect forms against CSRF (Cross-Site Request Forgery) attacks. These cookies are destroyed when you close your browser or upon session expiry.

4. IP Addresses and Access Logs

Upon accessing this website, our web server automatically processes technical connection data, including your IP address. This processing is technically required to serve the webpage, mitigate cyber-attacks, and prevent automated abuse (e.g. bots). Server logs are automatically purged in accordance with data retention regulations.

5. Third-Party Resources (Google Fonts)

This website integrates web typography ("Google Fonts") provided by Google LLC. To display these fonts, your web browser establishes a direct connection to Google's servers, transmitting your IP address to Google. For more details, consult Google's Privacy Policy at: policies.google.com/privacy.
